【Sokusel アプリの使い方】

1. アプリの起動
   index.html をダブルクリックしてブラウザで開いてください。
   
2. Googleドライブ連携
   画面上の「認証」ボタンを押すと、Googleドライブへのアクセス許可が求められます。
   許可すると、「sokusel」というフォルダがGoogleドライブに作成され、そこに問題データ(questions.json)と成績データ(stats.json)が保存されます。
   次回からは自動的にドライブのデータを読み込みます。
   
3. 問題の作成・編集
   「編集」ボタンから問題を作成・編集できます。保存するとGoogleドライブにも反映されます。

4. GitHubへのアップロードについて
   現在、パソコン上に「Gitリポジトリ」を作成しました（コミット済みです）。
   ブラウザでGitHubを開き、新しいリポジトリを作成してから、以下のコマンドをコマンドプロンプトやターミナルで実行するとアップロードできます：
   
   git remote add origin [あなたのGitHubリポジトリURL]
   git branch -M main
   git push -u origin main

【トラブルシューティング】
・「止まる」場合
   インターネットに接続されていることを確認してください（Googleのシステムを利用するため）。
   Googleの認証画面がポップアップブロックされていないか確認してください。
